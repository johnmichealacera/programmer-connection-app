# Programmer Connection App 🚀

A next-generation social network for programmers that focuses on **genuine connections** rather than vanity metrics. Built with modern web technologies and designed for authentic developer relationships.

## 🌟 Core Philosophy

**No followers. No likes. No vanity numbers.**

This app is built on the principle that meaningful connections happen through:
- **Resonance** - Finding people who share your energy and interests
- **Flow** - Natural, effortless interactions
- **Belonging** - Being part of meaningful micro-communities

## ✨ Key Features

### 🎯 Instant Onboarding
- **3 simple questions** to understand your programming energy
- **Automatic pairing** with like-minded developers
- **No profile perfection** required - just authentic responses

### 🔗 Resonance Actions
- **"Resonates"** - Show genuine connection without metrics
- **"Thanks"** - Express gratitude for shared insights
- **Private feedback** - No public counters or vanity

### 👥 Automatic Group Formation
- **Micro-groups** (2-6 people) based on shared interests
- **Ephemeral or long-term** depending on activity
- **Purpose-driven** conversations and collaboration

### 📱 Flow Feed
- **Curated content** instead of infinite scroll
- **Minimal effort** interaction buttons
- **Meaningful prompts** that encourage real connections

### 💬 Connection Buttons
Pre-suggested ways to connect:
- "Looking for a study buddy in Python"
- "Excited about AI in education"
- "Need feedback on my side project"
- "Want to collaborate on open source"

## 🛠️ Tech Stack

- **Frontend**: Next.js 14 + TypeScript
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **State Management**: React Hooks
- **Design**: Mobile-first, responsive design

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone <your-repo-url>
   cd programmer_connection_app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

## 🎮 How to Use

### 1. Onboarding Experience
- Answer 3 resonance-based questions about your programming mood, focus areas, and what excites you
- The app automatically analyzes your responses and finds compatible developers
- Get introduced to your "resonance group"

### 2. Dashboard
- **Quick Connections**: Pre-suggested ways to connect with others
- **Flow Feed**: Curated content from your community
- **Resonance Stats**: Track meaningful connections (not vanity metrics)

### 3. Group Interactions
- Join micro-groups based on shared interests
- Use "Resonates" and "Thanks" for authentic feedback
- Participate in focused, purpose-driven conversations

## 🎨 Design Principles
